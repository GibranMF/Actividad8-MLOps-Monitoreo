# Actividad 8 - Monitoreo y Observabilidad de un Modelo de Machine Learning

## Descripción

Este proyecto corresponde a la Actividad 8 de la materia Gestión de Proyectos de Inteligencia Artificial.

Se implementa una estrategia de observabilidad para un modelo de Machine Learning en producción utilizando:

- MLflow
- Python
- Métricas
- Logs
- Trazas
- Alertas inteligentes
- Runbooks
- Detección de Data Drift

## Modelo Utilizado

Random Forest para aprobación de créditos.

## Estructura del Proyecto

/docs
Documentación técnica y PDF final.

/screenshots
Capturas de MLflow, dashboards y evidencias.

/src
Scripts de monitoreo y automatización.

/notebooks
Análisis y simulaciones.

/config
Configuración de alertas.
